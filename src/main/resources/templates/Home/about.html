Revolutionizing Event Management

SecurePass is a cutting-edge web application designed to streamline event invitation management with advanced facial recognition technology.

Our platform allows users to register with personal details and a facial photo, which is processed using OpenCV and YOLO for secure storage in a MySQL database. During events, administrators can verify attendees’ identities in real-time, ensuring a safe and efficient experience without relying on traditional tickets or QR codes. Built with Spring Boot, JWT authentication, and a user-friendly interface, SecurePass prioritizes security, simplicity, and performance.
